# automated-iphone-calender

Einfache GitHub-Pages-Seite zum Abonnieren eines `.ics`-Kalenders.

## Dateien

- `index.html`: zeigt einen Button, der den Abo-Link für iPhone (`webcal://...`) anzeigt.
- `calendar.ics`: Kalenderdatei, die von abonnierenden Geräten regelmäßig aktualisiert wird.

## Nutzung

1. GitHub Pages für dieses Repository aktivieren.
2. Die Seite öffnen und auf **Kalender-Link anzeigen** klicken.
3. Den `webcal://`-Link auf dem iPhone öffnen, um den Kalender zu abonnieren.

Wenn `calendar.ics` im Repository geändert wird, erhalten Abonnenten die aktualisierten Termine über das Kalender-Abo.
